## C Programming – Learning Journey

This repository contains my learning progress and practice programs in the C programming language.  
The course covers fundamental to advanced concepts required to build a strong foundation in C programming.

### Topics Covered
- Introduction to C Programming
- Compilation Process & Preprocessor Directives
- Tokens, Keywords & Data Types
- Variables, Constants & Input/Output
- Operators, Expressions, and Operator Precedence
- Branching and Looping Statements
- Arrays (1D, 2D, and Multidimensional)
- Searching and Sorting Techniques
- Strings and String Manipulation
- Functions and Parameter Passing
- Recursion
- Pointers and Pointer Arithmetic
- Advanced Pointer Concepts

### Objective
The objective of this repository is to practice C programming concepts through examples and programs, and to strengthen problem-solving and programming skills.
